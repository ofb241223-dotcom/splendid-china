# 🇨🇳 锦绣中华 · 盛世河山 (Splendid China)

> **“江山如此多娇，引无数英雄竞折腰。”**

## 📖 项目简介 (Introduction)

**锦绣中华** 是一款基于 Web 技术构建的沉浸式交互网页。本项目旨在通过数字化的艺术形式，重现中国大好河山的壮丽轮廓。

从开场的中式大门徐徐展开，到指尖滑过各省份时的流光溢彩，再到详尽的人文地理介绍。我们摒弃了复杂的后端依赖，将中国 **34个省级行政区**（含港澳台）的历史、文化、美景与美食浓缩于代码之中，由衷地表达对祖国的热爱。

## ✨ 核心功能 (Features)

*   **⛩️ 震撼开场动画**：
    *   纯 CSS3 实现的中式红门/画卷开启特效。
    *   伴随页面加载，大门向两侧平滑拉开，营造庄重的仪式感。
*   **🗺️ 全域交互地图**：
    *   基于 **Apache ECharts** 渲染的高清矢量地图。
    *   **智能简称**：地图标签自动显示省份简称（如“湘”、“京”、“台”）。
    *   **高亮聚焦**：点击任意省份，地图自动高亮选中区域，视觉体验极佳。
*   **📜 详尽人文数据**：
    *   **无一遗漏**：内置中国 34 个省级行政区的深度文案，绝无“占位符”。
    *   **打字机特效**：介绍文字以打字机效果逐字显现，不仅是阅读，更是品味。
*   **🎨 国潮视觉美学**：
    *   **配色**：中国红 (#C41E3A) 与 琉璃金 (#FFD700) 的经典碰撞。
    *   **纹理**：背景采用淡雅宣纸纹理，增添历史厚重感。
    *   **交互**：鼠标移动自带金色粒子拖尾，寓意“星火燎原”。

## 📂 项目目录结构 (Project Structure)

本项目为纯静态架构，无任何复杂的构建工具，结构清晰：

```text
splendid-china/
├── index.html    # 🚀 项目主入口 (包含核心逻辑与所有数据)
├── china.json             # 🌏 地图数据源 (GeoJSON)
└── README.md              # 📘 项目说明文档
```
## 🚀 如何在本地运行 (Local Setup)
由于浏览器安全策略（CORS），请勿直接双击 splendid-china.html 打开，否则无法加载 china.json 地图数据。

* 使用 VS Code Live Server
* 使用 VS Code 打开 splendid-china 文件夹。
* 安装插件 Live Server。
* 在 splendid-china.html 文件上右键，选择 "Open with Live Server"。
